Estudo sobre NGRX, o Redux para Angular
================================

Descrição
---------
No estudo sobre NGRX, implementei um crud de lista de compras com gerenciamento de estado (estilo Redux). Contém componentes para adicionar itens e visualizar a lista, um serviço para operações e arquivos de store (actions, reducer, selectors, effects, state). 

O que fiz
--------
- Componentes: adicionar-ingrediente, lista-compras
- Serviço: lista-compras.service
- Store: lista-compras.action.ts, lista-compras.reducer.ts, lista-compras.effects.ts, lista-compras.selectors.ts, lista-compras.state.ts

Como rodar
----------
1. Instale dependências:

   npm install

2. Inicie a aplicação:

   npm start

Estrutura principal
------------------
- src/app/adicionar-ingrediente/
- src/app/lista-compras/
- src/app/store/
